Pizarra Digital

Nos permite realizar trazos y formas como: cuadrados, rectangulos, elipses, circunferencias. Además podemos descargar una imagen JPEG de lo que se haya escrito en ella.

Caja de Herramientas

Aquí podemos encontrar las herramientas necesarias para utilizar
Entre ellas:

  • Figuras
  • Opciones
  • Colores
  • Acciones

Figuras

En esta sección se encuentra diferentes figuras para dibujar

  • Rectángulo dibuja rectángulos manteniendo presionado el click.
  • Circulo dibuja cículos manteniendo presionado el click.
  • Triángulo dibuja triángulos manteniendo prsioando el click.
  • Relleno si está desactivado se dibujan solo las lineas de la figura, si está activado se dibujan las figuras con relleno(pintadas).

Opiones

En este apartado están disponible las opciones de:

  • Rayar dibuja lineas manteniendo presionado el click y moviendo el mouse.
  • Borra borra por area lo que hayas hecho por error manteniendo presionado el click y moviendo el mouse.
  • Barra de grosor de linea permite regular el grosor de la linea.

Colores

En esta sección podremos cambiar el color con el que se va a dibujar las figuras o las lineas, se presentan 4 primeras opciones con colores fijos. Sin embargo, si necesitas un color personalizado lo puedes obtener en la ultima opción de color.

Acciones

Podemos realizar 2 opciones:

  • Limpiar Pizarra borra todo el contenido de la pizarra.
  • Guardar Imagen se descarga una imagen en formato PNG del contenido de la imagen.

Zona de dibujo

En esta parte podremos realizar todos los dibujos que deseemos con las herramientas ya mencionadas anteriormente.

Simulador 2D

Esta herramienta nos ayuda a comprender la parte teórica con la práctica através de diferentes objetos (autos), los cuales se mueven y recorren distancias a escala de los datos que se ingrsen y se calculen en la sección de datos entre sus principales carácterísticas tenemos

  • Crear hasta 4 objetos.
  • Realizar calculos de los objetos siempre y constee de una incógnita.
  • Conversor de unidades al momento de realizar el cálculo de la incógnita.
  • Una barra de estado, la cual nos informa cuando estaremos listos para emepezar con la simulación.
  • La sección para la simulación mediante la representación gráfica del movimiento de los objetos.

Generar objetos

Podemos crear entre 1 y 4 objetos para poder manipular sus datos y posteriormente verlas representados de forma gráfica en la simulación.
Una vez seleccionado el número de objetos damos click en Generar

Datos de los objetos

En esta parte nosotros debemos llenar almenos 2 datos de cada objeto para poder conitnuar y el tercer dato sin llenar se calculará al presionar el botón Calcular.
El caso de no llenar almenos los 2 datos, no se podrá calcular el valor del dato que deseamos conocer.

Unidades de medida

Las unidades disponibles son:

    Velocidad

  • km/h (kilómetro por hora)
  • km/min (kilómetro por minuto)
  • km/s (kilómetro por segundo)
  • m/h (metro por hora)
  • m/min (metro por minuto)
  • m/s (metro por segundo)
  • cm/h (centrímetro por hora)
  • cm/min (centrímetro por minuto)
  • cm/s (centrímetro por segundo)

    Distancia

  • km (kilómetro)
  • m (metro)
  • cm (centímetro)

    Tiempo

  • h (hora)
  • min (minuto)
  • s (segundo)

Una vez seleccionada la unidad de medida al momento de realizar el cálculo el programa realizará las conversiones de unidades necesarias para mostrar la respuesta la unidad de medida seleccionada.

Botón calcular

Al presionar calcular si no se cumplen con las indicaciones ya mensionadas en secciones anteriores se mostrará un mensaje con la onservación correspondiente, caso contrario, se procede a realizar el cálculo

Barra de estado

Esta barra inicialmente se encontrará en color rojo el cual nos indica que no hay ningún objeto creado y no está lista para la simulación; el color amarillo nos indica que está casi listo para la simulación, es decir, falta que se llenen los datos necesarios para poder estar listos. Finalmente el color verde nos quiere decir que se llenaron todos los datos suficientes para poder empezar con la simulación, lo que implica que necesariamente los datos de distancia y velocidad estén definidas.

Gráfico de simulación

Aquó podremos observar el estado inicial de los objetos al momento de crearlos. Luego al dar click en el play se podrá observar su movimiento y su recorrido.

Botón play

El boton play inicialmente estará bloqueado, asta que la barra de estado se cuentre en color verde, lo que da señal de que todo está listo para realizar la simulación.

Generador de gráficos

Nos permite generar gráficos de Distancia(m) versus Tiempo(s) de forma automática o de manera personalizada. Entre sus principales carácterísticas tenemos:

  • Graficar hasta 10 puntos o coordenadas.
  • Presentación de soluciones a cada segmento de la gráfica.
  • Ingreso de datos para generar gráficos personalizados
  • Botón de descarga, el cual nos permite descargar el gráfico generado

Modo de generar el gráfico

Aquí seleccionaremos la forma en que se va a generar el gráfico de forma automática o de manera personalizada.

Numero de datos automático

Si elejimos la creación de manera automática, tendremos que definir el número de datos que conformará el eje de las "x" y de igualmenra del deje de las "y".
Para esto debemos tomar en cuenta que el número de datos del eje de las "y" no deben superar al número de datos del eje de las "x".

Presentación de gráfica

Una vez que se definan los datos pertinentes para la gráfica lo veremos representado en esta parte.
En el gráfico se detallan la etiqueta de cada punto en al gráfica, las guías de los puntos y los valores de cada punto.

Ver solución

Se debe seleccionar e segmento del cual se desea ver a solución.
Se podrá observar las coordendas implicadas, la formula que se utiliza para encontrar la fórmula y finalmente la aplicación de la fórmula con su proceso.

Número de puntos personalizado

Se debe escoger el número de puntos o coordenadas que tendrá el grádico.

Campos de datos

Aquí debemos llenar todas las coordenadas para generar el gráfico Recuerda trabajar con valores positivos, puesto que si se trabajan con valores negativos la gráfica no se apreciará correctamente.

Botón generar

Una vez que todo se encuentre correctamente al dar click se generará el gráfico, caso contrario se mostrará un mensaje con la observación pertinente.

Descargar Imagen

Si deseamos guardar la imagen, desde este apartado se lo puede descargar.
